# react-hooks-demo

Live demo for [react-idle-detect](https://www.npmjs.com/package/react-idle-detect) and [@auraqule/react-network-status](https://www.npmjs.com/package/@auraqule/react-network-status).

## Run locally

```bash
pnpm install
pnpm dev
```

## Deploy to Vercel

```bash
npx vercel
```

Or connect the GitHub repo on [vercel.com](https://vercel.com) for auto-deploys.

## Packages used

- `react-idle-detect` — detects user inactivity
- `@auraqule/react-network-status` — monitors online/offline state
