# react-hooks-demo

Live demo for [react-idle-detect](https://www.npmjs.com/package/react-idle-detect) and [@auraqule/react-network-status](https://www.npmjs.com/package/@auraqule/react-network-status).

## Demo

Visit the live demo: [https://react-packages-demo.vercel.app/](https://react-packages-demo.vercel.app/)

## Features

### react-idle-detect

- Detects user inactivity after configurable timeout
- Shows real-time idle/active state
- Visual timer bar counting down to idle
- Event log showing idle/active transitions
- Manual reset button

### @auraqule/react-network-status

- Monitors online/offline status
- Displays connection type (WiFi, Ethernet, Cellular, etc.)
- Shows effective connection speed (4G, 3G, 2G, slow-2G)
- Displays downlink speed (Mbps) and RTT (ms)
- Data saver preference detection
- Manual refresh button
